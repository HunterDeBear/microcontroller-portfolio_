Project 03

Smart Perimeter Detection System

A laser-based security prototype using an LDR receiver, ADXL accelerometer, OLED display, and active buzzer alert tones.

Project Status

Prototype

Skills Used

STM32 LDR Laser ADXL OLED Security

Overview

This project is a smart perimeter security system designed to detect both intrusion and tampering. It uses a laser beam aimed at an LDR receiver to detect beam interruption, while an ADXL accelerometer monitors physical movement or tilt of the system.

Objective

  • Detect perimeter intrusion using a laser and LDR receiver.
  • Detect tampering using an ADXL accelerometer.
  • Display system state and alert category on an OLED.
  • Use buzzer tones to identify different alert conditions.
  • Prepare the system for future ESP32 wireless communication.

Components Used

  • STM32 microcontroller
  • Laser transmitter module
  • LDR receiver circuit
  • ADXL accelerometer
  • OLED display
  • Active buzzer
  • ESP32 planned for wireless expansion

Working Principle

The laser transmitter continuously shines onto the LDR receiver. When the beam is blocked, the LDR reading changes and the system detects this as an intrusion event.

The ADXL accelerometer adds a second security layer by detecting tilt or movement of the system hardware. The OLED display shows whether the alert is caused by intrusion or tampering, while the active buzzer produces different tones for alert categories.

Results and Insights

  • The system detected laser beam interruptions successfully.
  • The LDR receiver worked better when shielded from ambient light.
  • The accelerometer helped detect physical tampering.
  • The OLED made the system state clear and easy to understand.

Future Improvements

  • Add ESP32 wireless communication.
  • Send mobile notifications during alerts.
  • Add event logging and timestamps.
  • Design a stronger enclosure for outdoor testing.
