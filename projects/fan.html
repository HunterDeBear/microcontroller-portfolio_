Project 02

Temperature-Controlled Fan System

An ESP32-based thermal control system using temperature feedback, PWM fan control, MOSFET driving, and selectable cooling modes.

Overview

This project is an embedded temperature regulation system that automatically controls the speed of a 12 V brushless DC fan. The system uses temperature feedback from a DHT22 sensor and adjusts fan speed through PWM.

Objective

  • Measure real-time temperature using a DHT22 sensor.
  • Control fan speed automatically using PWM.
  • Implement Whisper, Standard, and Dynamic cooling modes.
  • Reduce sudden speed changes using smoothing.
  • Prevent fan stalling using a soft-start method.

Components Used

  • ESP32 Pico microcontroller
  • DHT22 temperature sensor
  • IRLZ44N MOSFET
  • 12 V brushless DC fan
  • I2C LCD display
  • Push buttons
  • WiFi interface

Working Principle

The DHT22 sensor measures the surrounding temperature. The ESP32 compares the measured value with a desired temperature range and calculates the required fan speed. A PWM signal is then sent to the MOSFET, which controls the 12 V fan.

Different operating modes use different proportional gains. Whisper mode focuses on quiet operation, Standard mode balances cooling and noise, and Dynamic mode provides faster cooling response.

Results and Insights

  • The fan speed changed smoothly based on temperature.
  • Each cooling mode produced a different response.
  • The system remained stable without major oscillations.
  • The soft-start feature helped prevent fan stalling.
