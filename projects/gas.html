Project 01

Smart Gas Monitoring System

An STM32-based gas monitoring and alert system with LCD output, Bluetooth transmission, buzzer alerts, and LED level indication.

Complete

STM32 ADC UART I2C LCD PWM Bluetooth

Overview

This project is an embedded gas monitoring system designed to detect gas concentration levels in real time. It uses an STM32 microcontroller to process analog sensor readings and provide immediate visual, audio, and wireless alerts.

Objective

  • Monitor gas concentration in real time.
  • Convert analog sensor voltage into estimated PPM values.
  • Display gas level and concentration on an LCD.
  • Transmit data wirelessly using Bluetooth.
  • Trigger LEDs and buzzer alerts based on gas level.

Components Used

  • STM32 Nucleo-F411RE microcontroller
  • MQ gas sensor
  • 16x2 LCD with I2C interface
  • HC-05 Bluetooth module
  • Active or passive buzzer
  • LED indicators
  • Push buttons for display modes

Working Principle

The gas sensor produces an analog output voltage that changes according to the detected gas concentration. The STM32 reads this voltage using its ADC, converts it into a digital value, and then estimates the gas concentration in PPM.

The processed value is displayed on the LCD, transmitted through the HC-05 Bluetooth module, and used to activate LEDs and buzzer alerts when the concentration reaches certain levels.

Results and Insights

  • The system responded clearly to changes in gas concentration.
  • The LCD successfully displayed gas level and PPM values.
  • Bluetooth communication allowed remote monitoring.
  • The buzzer and LEDs provided immediate local alerts.
